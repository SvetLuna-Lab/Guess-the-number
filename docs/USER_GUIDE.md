# Руководство пользователя

1) Программа попросит правую границу диапазона `n`.
2) Затем нужно вводить числа от `1` до `n`.
3) Программа будет подсказывать: больше/меньше.
4) После угадывания покажет количество попыток.
5) Потом спросит, хотите ли сыграть ещё раз.

Ответы на вопрос о повторной игре:
- да: `да / д / yes / y`
- нет: `нет / н / no / n`
