# Требования к проекту (конспект)

## Заголовок программы
1) Подключить модуль `random`
2) Сгенерировать случайное число
3) Вывести приветствие пользователю:
   `Добро пожаловать в числовую угадайку`

## Валидация ввода
Написать функцию `is_valid(text, right)`, которая возвращает `True`, если:
- `text` — целое число
- `1 <= int(text) <= right`

Иначе вернуть `False`.

Если ввод некорректен — выводить:
`А может быть все-таки введем целое число от 1 до N?`

## Основной цикл
- Запросить правую границу `n`
- Загадать число от `1` до `n`
- Угадывание в цикле:
  - меньше → `Ваше число меньше загаданного, попробуйте еще разок`
  - больше → `Ваше число больше загаданного, попробуйте еще разок`
  - равно → `Вы угадали, поздравляем!`
- После угадывания вывести количество попыток
- Предложить сыграть ещё раз (да/нет)
- На выходе вывести:
  `Спасибо, что играли в числовую угадайку. Еще увидимся...`
